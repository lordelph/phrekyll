Installation

 

Depending on your purposes there are several ways to install Phrekyll:

If you plan to extend Phrekyll it is suggested to fork official repository, otherwise proceed with installation via Composer. Should you need raw sources, you can have them too.

System Requirements

  • PHP 5.3+

Pre-Built

You can download a prebuilt phar file of the latest release from https://github.com/lordelph/phrekyll/releases/latest For example, here's how you can download the latest release and install it globally... 
	curl -L -O https://github.com/lordelph/phrekyll/releases/download/v.0.9.0/phrekyll.phar
	chmod 755 phrekyll.phar
	sudo mv phrekyll.phar /usr/local/bin/phrekyll
	sudo ln -s /usr/local/bin/phrekyll /usr/local/bin/phr

Use Composer

If you do not have Composer installed, install it first: 

  curl -s https://getcomposer.org/installer | php

Once Composer is installed, proceed with Phrekyll installation: 

  php composer.phar create-project lordelph/phrekyll phrekyll && cd phrekyll

You can then symlink phr executable located in bin folder:

  sudo ln -s `pwd`/bin/phrekyll.php /usr/local/bin/phr

That's it! Read Getting Started guide for a quick start.
