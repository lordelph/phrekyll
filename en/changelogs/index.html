  • Content Providers added

  • Phrekyll Bundles repository created for user-contributed bundles.
  • Better Front Matter extraction (bug fixed)
  • Phrekyll started using Phing for managing bundle builds.
  • Table of Contents added to documentation pages ( implementation)
